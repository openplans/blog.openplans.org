Mjumbe will talk at the May JavaScript Meetup

Posted on May 30, 2013 by Anthony Denaro

Mjumbe will speak about using Backbone.js at the the May JavaScript Meetup in Philly.

Backbone.js is a super light-weight framework that allows you to structure your Javascript code in an MVC (Model, View, Controller) fashion. Backbone does well under a certain set of assumptions about the APIs it interacts with, but can also be extended to work with APIs that violate these assumptions. I’ll go over backbone’s assumptions about data, and some strategies to help Backbone work with APIs that don’t conform to these assumptions.

Thursday, May 30, 2013 @ 7pm
Quorum at the Science Center
3711 Market Street, Floor 8, Philadelphia, PA (map)
